## Project Overview
AI Medical Assistant is a web application that provides preliminary health guidance based on user symptoms using **AI and Natural Language Processing (NLP)**.

## Features
- Symptom-based health guidance.
- Conversational interface for better user interaction.
- Backend powered by **Flask**.
- Machine learning model to analyze user symptoms and provide suggestions.

## Tools & Technologies
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Machine Learning & NLP:** scikit-learn, NLTK
- **Others:** REST APIs for symptom data
